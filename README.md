# Обрезка ссылок с помощью VK

Данный проект помогает сокращать ссылки с помощью сервиса Вконтакте(далее VK), а также посчитать кол-во переходов по уже сокращенной таким образом ссылке.

### Как установить

Должен быть установлен [Python 3](https://www.python.org/downloads/).

Затем используйте `pip` (или `pip3`, если существует конфликт с Python2) чтобы установить необходимые для работы программы пакеты Python:

```
pip install -r requirements.txt
```

### Как работает

Программа принимает обязательный параметр `url` - Ваша ссылка, которую хотите сократить. Полученная ссылка сокращается на сервисе VK, после чего выводится сокращенная ссылка. 

Если программе передается ссылка, которая уже является сокращенной через сервис VK, тогда подсчитывается и выводится кол-во переходов по этой сокращенной ссылке.

### Демонстрация работы

В консоли указываем директорию, в которою разместили данный скрипт:
```
cd *путь к скрипту*
```

После чего запускаем скрипт `main` передавая вместе с ним Вашу ссылку:

```
python main.py *Ваша ссылка*
```

После этого Ваш выведется ссылка, сокращенная с помощью VK.

Если же Вы указали ссылку уже сокращенную с помощью VK, тогда скрипт выведет кол-во переходов по ней.
